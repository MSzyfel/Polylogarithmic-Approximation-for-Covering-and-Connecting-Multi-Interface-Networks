\begin{abstract}
We consider the problem of covering/connecting a multi-interface network with a minimum cost assignment of interfaces to vertices. In the covering problem we demand that every edge is covered by at least one common interface assigned to its endpoints. In the connection problem we are given a set of terminals $U\subseteq V$ and we demand that the subgraph of activated edges spans all of them.
We give an $O\br{\log m}$ approximation algorithm for the min-max cost criterion. The algorithm is based on rounding the optimal solution of a natural, set-cover-style linear programming relaxation of the problem. We also show how to use this LP to obtain a $k$-approximation algorithm, where $k$ is the number of interfaces. Importantly, both of the aforementioned procedures work even when the cost of activating an interface depends on the interface as well as the vertex to which it is assigned.
This improves the previously best known $O\br{\ln\Delta+b\cdot\min\brc{\ln \Delta,c_{\text{max}}}}$-approximation, where $b$ is a certain parameter of the input instance which can be as large as $\Omega\br{n}$ and $c_{\text{max}}$ is the maximum cost of activating an interface and $O\br{k}$-approximation for the uniform cost case [Algorithmica 2012]. 
% For min-sum covering this improves the previously best known $O\br{\sqrt{n}\cdot\log n}$-approximation and generalizes the $O\br{k}$-approximation [Wireless Networks 2009]. 
For the connection problem we devise an $O\br{\log^2 m}$-approximation algorithm, which is the first non-trivial approximation algorithm for this problem. The algorithm is based on a similar LP relaxation with additional cut constraints to ensure connectivity of the terminals. 
% We also give combinatorial algorithms with $O\br{\Delta}$ guarantees for both min-sum and min-max covering as well as min-sum connection thus improving on the previously known $O\br{\Delta\cdot c_{\text{max}}}$ for the min-max covering.

% We complement these results by showing that both the min-sum covering and connecting cannot be approximated within a factor of $\br{1-o\br{1}}\cdot \ln n$ even on stars, unless $P=NP$. We also show, by an approximation-preserving reduction to the min-sum covering assignment problem on bipartite graphs with unit costs, that min-sum covering cannot be approximated within a factor of $o\br{\log \Delta}$ unless $P=NP$.
\end{abstract}
