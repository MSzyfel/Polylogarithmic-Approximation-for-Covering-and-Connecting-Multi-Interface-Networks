\begin{abstract}
We study problems in multi-interface networks of wireless devices that can communicate if they are close enough and activate a common interface. We consider the heterogeneous setting, where activating an interface induces a cost that depends on the type of interface and on the vertex that activates it. The goal is to cover or to connect the network at minimum cost. In the coverage problem, every connection defined in the network must be established, while in the connectivity problem, groups of terminals specified in the input should be connected.

For the coverage problem, we present the first approximation algorithms for the heterogeneous setting: we provide a  $O\br{\log m}$-approximation algorithm for the min-max cost criterion (minimizing the maximum cost in a vertex), and we provide a $k$-approximation algorithm, where $k$ is the number of interfaces, for both the min-max and the min-sum cost criterion (minimizing the sum of costs). This $k$-approximation algorithm generalizes a similar results in the homogeneous setting, where costs depend only on the type of interface, to the heterogeneous setting. 
For the connectivity problem, we present the first approximation algorithm for the min-max cost criterion: we provide a $O\br{\sqrt{n}\cdot\log n}$-approximation.
Our algorithms rely on a randomized rounding of the solution a natural, set-cover-style linear programming relaxation.
