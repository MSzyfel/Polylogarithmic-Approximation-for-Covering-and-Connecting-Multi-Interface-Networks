\section{Conclusion} \label{sec:conclusion}

In this paper, we have provided first polylogarithmic approximation algorithms for the \connectivity problem, which is a fundamental problem in covering multi-interface networks. We have also given an $O\br{\log m}$-approximation algorithm for the \coverage problem, which is almost tight since the problem is NP-hard to approximate within a factor of $o\br{\log \Delta}$, even if the input is a star and all costs are uniform. This inapproximability carries on also for the \connectivity problem, since on stars, \connectivity with one terminal set $U_1=V$ is equivalent to \coverage. The remaining question is to close the gap between the $O\br{\log^2 m}$-approximation algorithm for \connectivity and the $O\br{\log m}$ inapproximability. We conjecture that indeed $O\br{\log^2m}$ might be best possible, since the problem shares a lot of similarities with the \textsc{Group Steiner Tree} which cannot be approximated better~\cite{PolylogarithmicInapproximability}.
